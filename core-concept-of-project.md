Attendance & Academic Management Portal – Core Context

1. Product Overview

This application is a web-based Attendance & Academic Management Portal designed for schools.
Its primary goal is to enable schools to evolve in the modern era by seamlessly tracking student attendance, academic performance, and staff effectiveness within a centralized system.

The system is intended to be used daily by school leadership, teachers, students, and parents.

2. Target Institutions

Medium-sized schools

International schools

The product is designed to feel professional, reliable, and institutional rather than experimental or overly complex.

3. Core Philosophy

Simplicity over feature overload

Role-based clarity

Trust and reliability during school hours

Centralized data as a single source of truth

Long-term academic data continuity

4. Platform Scope

Web application only

Responsive design to support desktop and mobile browsers

No native mobile applications in the current scope

5. User Roles & Access Model
5.1 Super Admin (Directors / Principals)

Full visibility and control across the school

Manages academic structure, users, and calendar

Creates and sends official announcements

Views high-level attendance and academic summaries

Monitors teacher performance based on student outcomes

5.2 Teachers

Access limited to assigned classes and students

Mark attendance

Create assessments and enter marks

View class-level academic performance

5.3 Students

View-only access to their own:

Attendance records

Academic performance

Academic calendar

5.4 Parents

Interactive access (no alerts)

Can:

View their child’s attendance and academic performance

Apply for leave

Acknowledge official notices

Can only access data related to their own child

6. Attendance Management

Hybrid attendance model:

Daily attendance (default)

Period-wise attendance (optional per school)

Attendance states:

Present

Absent

Late

Attendance marked only by teachers

Attendance data preserved across academic years

7. Academic Management

Fully flexible assessment framework

No hardcoded exam structures

Teachers or admins define:

Assessment name

Maximum marks

Applicable classes

Marks are logged within the portal

Initial scope includes only basic reports

8. Reporting Scope (Initial Phase)

Attendance percentages

Marks summaries

Simple averages

Individual student views

Class-level summaries

School-level overviews for Super Admin

Advanced analytics and insights are out of scope for the initial version.

9. Academic Calendar

Single academic calendar shared across all users

Created and managed only by Super Admin

Read-only access for teachers, students, and parents

10. Communication Model

One-way, official communication only

Announcements created by Super Admin

Target audience selection:

Entire organization

Teachers

Parents

Messages delivered via WhatsApp using stored phone numbers

Portal remains the source of truth; WhatsApp is the delivery channel

11. Authentication & Security

Username and password authentication for all users

Strict role-based access control

Teachers and parents cannot access unrelated data

Data integrity and privacy are mandatory

12. Data Continuity

Longitudinal data model

Student, teacher, attendance, and academic records persist across academic years

Historical data remains accessible to Super Admin

13. Deployment Model

Single-school deployment in the current phase

Architecture should remain future-ready for multi-school expansion

14. Non-Goals (Explicitly Out of Scope)

Fee management

Two-way parent-teacher chat

Push notifications or alerts to parents

Native mobile apps

Advanced AI-based analytics

15. Future Expansion (Awareness Only)

Multi-tenant (multi-school) support

Advanced performance insights

Parent alerts (optional)

Email/SMS communication fallback

Mobile applications

16. Success Criteria

Teachers can mark attendance quickly and reliably

Principals gain clarity without complexity

Parents trust and understand their child’s academic status

The system becomes part of daily school operations
